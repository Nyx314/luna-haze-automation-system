# Luna Haze Automation System

Automation system built for Luna Haze Smoke Shop (Miami).  
This project consolidates the workflow architecture, AI prompts, reactivation flows, pickup-order integration, and SEO optimization logic used to increase customer visits, streamline orders, and revive inactive customers.

## Objectives
- Reactivate past customers through SMS + email sequences.
- Automate pickup-order processing from the website into Airtable.
- Generate consistent branded content using AI workflows.
- Improve Google visibility (SEO + AIO content pipeline).

## System Overview
1. Customer enters phone/email → added to Airtable.
2. Automated tagging → sends correct SMS sequence.
3. Abandoned interest triggers follow-ups.
4. Website pickup orders trigger notifications + fulfillment dashboard.
5. SEO content prompts generate product descriptions, listings, and Google-optimized posts.

## Tools Used
- Brevo (SMS + Email)
- Airtable (Order + Customer DB)
- Website admin portal
- AI prompt automation
- Make / Zapier for JSON workflows

## Impact
- Increased customer return rate.
- Higher pickup-order completion.
- Faster order fulfillment through Airtable interface.
- Consistent SEO-optimized content output.
